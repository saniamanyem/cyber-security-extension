**Cybersecurity for the Modern Digital Building**
Integration of Building Systems and Cybersecurity Controls 

**Overview**
Modern digital buildings contain interconnected physical, operational, and information technology assets. While building ontologies such as ASHRAE 223 effectively model physical systems, equipment, spaces, and connectivity, they do not explicitly represent cybersecurity controls.

**Project goal:
**The project aims to create a cybersecurity-based building knowledge graph by overlaying cybersecurity concepts onto an existing ASHRAE 223 building model. The result will connect physical building infrastructure, operational technology, building automation systems, and cybersecurity controls within a unified framework. The project will also evaluate how semantic building models can be used as a foundation for cybersecurity reasoning and decision-making in digital buildings. 

The foundation of the model will consist of: 
ASHRAE 223 for building system representation
PAE-derived operational use cases
MITRE D3FEND cybersecurity vocabulary
IFC identifiers (GUIDs) linking digital building assets to semantic entities

**Research question:**
How can cybersecurity concepts be semantically integrated with ASHRAE 223 building models to create a unified knowledge graph for digital buildings, and what cybersecurity value does this model provide for digital building operations?
